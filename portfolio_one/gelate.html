概要

模擬課題にて、Webサイトを作成。デザインからコーディングまで行いました。

期間
2019/12/2~2019/2/20(３週間)
使用ツール
Photoshop / Illustrator / Figma / HTML / CSS
制作物
Webサイト

●依頼内容

新宿駅近くのビルにて「Gelateria MAKO」のWebサイト制作(模擬)。 ターゲットは都内に住む１LDKで一人暮らしをする28歳女性会社員。趣味はカフェ巡りとヨガでシンプルな物を好む。

●プロセス

コンセプト：身体にやさしい本格ジェラードで、上質な一時を。 コンセプトとクライアントの特徴、ユーザーの特徴、競合他社から設計を行いました。

●デザイン

・ターゲットである女性のシンプルな物を好む。 　→Webのデザインをシンプルに。

・コンセプトから色の配色を高級的に、イメージ的に重くなりすぎないように。
