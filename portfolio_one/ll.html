概要

最終課題にて、Webサイトを作成。クライアントのヒアリングからデザイン、コーディングまで行いました

期間
2019/12/23~2020/1/16(３週間)
使用ツール
Photoshop / Illustrator / Figma / HTML / CSS
制作物
Webサイト

●依頼内容

新規事業コワーキング「Light Lounge」の店舗紹介のためのWEB制作。(模擬)。

●目的

店舗紹介、店舗への集客。

●イメージ

女性が気軽に来店できる場所、都会的、開放感

●ターゲット層

２０代〜３０代の男女。オシャレ、しっかりしており、やりたいことがある起業家。

●ヒアリング

現在のコワーキングスペースを利用している理由を調べたところ、料金が安いことが 一番の理由に上がった。

さらに、ドロップインの項目で検索されていることがわかった。

クライアンにヒアリング時に聞いた時に、都会的で開放感があり女性でも気軽にこれるイメージと強みである 料金の設定の安さ、また利用者は料金の安さを第一に求めていることがわかりました。
